# PimpBunny

Add artist pages to `artists.txt`, one URL per line:

```text
https://pimpbunny.com/onlyfans-models/belle-delphine-leaks/
...
```

After that, download the videos with:

```bash
python download.py
```

All stream files and downloaded videos are saved in the `artists` folder.
